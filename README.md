# 🚀 Meu Portfólio Pessoal

Este é o repositório do meu portfólio pessoal, onde apresento meus projetos, competências técnicas e trajetória acadêmica. O objetivo deste site é servir como um cartão de visitas digital, demonstrando minha capacidade de desenvolvimento de software e a diversidade de projetos que já desenvolvi.

## 🛠️ Tecnologias Utilizadas

Para a construção deste portfólio, utilizei as seguintes tecnologias:

- **HTML5**: Estruturação semântica do conteúdo.
- **CSS3**: Estilização, layout responsivo e animações.
- **JavaScript (ES6+)**: Interatividade e manipulação dinâmica de elementos.
- **Git & GitHub**: Controle de versão e hospedagem do código.

## 🌟 Destaques do Portfólio

No meu portfólio, você encontrará diversos projetos que demonstram diferentes competências, tais como:

- **Desenvolvimento Fullstack**: Projetos integrando Frontend com Backend e Bancos de Dados (MySQL).
- **Integração de APIs**: Consumo de APIs públicas (PokeAPI, SWAPI, etc.) para exibição de dados em tempo real.
- **Lógica de Programação**: Implementação de algoritmos (como a Busca Binária Animada).
- **UI/UX**: Foco em interfaces limpas, modernas e responsivas para diferentes dispositivos.
